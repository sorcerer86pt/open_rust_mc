# Conclusion:
# The CE adjoint photon slab walker (	ransport::adjoint_photon) is correct
# in the Boltzmann-adjoint sense — analog track-length tally on adjoint
# Compton + self-adjoint Rayleigh + absorption-as-source termination. But
# the resulting ψ̂*(z) profile is roughly symmetric with a mid-slab peak
# (spread by the symmetric direction sampling after each scatter), NOT
# the monotone gradient toward the detector that the RR-CADIS adjoint
# diffusion solve produces. With max/min ≈ 1.5× and a default ratio=5
# WW band, almost no voxel transition triggers splitting or roulette,
# so the transport runs are bit-identical to analog (within seed).
#
# RR-CADIS remains the recommended source of the shield_slab CADIS map.
# The CE adjoint walker is plumbed end-to-end (binary
# djoint_photon_cadis_slab writes the same JSON schema) so the wiring
# is in place for future work. Two natural follow-ups:
