# Benchmark Results

Generated by `scripts/run_rank_sweep.ps1`. Committed with the paper.

## Files

- `rank_sweep_godiva_*.csv` — Godiva bare sphere, SVD vs Table at ranks 1-6
- `rank_sweep_pwr_*.csv` — PWR pin cell, SVD vs Table at ranks 1-6
- `rank_sweep_gpu_godiva_*.csv` — GPU Godiva at ranks 1-6
- `rank_sweep_gpu_pwr_*.csv` — GPU PWR at ranks 1-6

## Columns

| Column | Description |
|--------|-------------|
| rank | SVD truncation rank |
| svd_k | SVD mode k_eff |
| table_k | Pointwise table k_eff (ground truth) |
| svd_table_gap_pcm | |svd_k - table_k| / table_k × 10^5 |
| svd_delta_exp_pcm | |svd_k - 1.0| × 10^5 (Godiva only) |
| svd_nsp / table_nsp | nanoseconds per particle |
| speedup | table_nsp / svd_nsp |

## Reproducing

```powershell
# Quick (3 seeds, reduced particles, ~30 min)
.\scripts\run_rank_sweep.ps1 -Quick

# Full paper quality (5 seeds, 50k Godiva / 20k PWR, ~2 hours)
.\scripts\run_rank_sweep.ps1

# With GPU benchmarks
.\scripts\run_rank_sweep.ps1 -GpuGodiva -GpuPwr
```
